# sentry-v29-yc-k8s-elastic

Проект развёртывания Sentry v29.x в Yandex Cloud на Kubernetes с **nodestore в Elasticsearch 9.x (ECK)** (пакет [sentry-nodestore-elastic](https://pypi.org/project/sentry-nodestore-elastic/)), а также Kafka, ClickHouse, PostgreSQL и Redis в составе инфраструктуры.

Подробные шаги установки: [readme.md](readme.md). Elasticsearch и кастомный образ Sentry: [elasticsearch.md](elasticsearch.md).
